# Marlvin Goremusandu - Portfolio

A modern, minimalist portfolio website showcasing my work as a software engineer, researcher, and builder.

## About

This portfolio highlights my journey from Zimbabwe through cloud engineering, AI research, and blockchain innovation. It features my projects, experience, and technical expertise in building systems that blend AI, blockchain, and design thinking.

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Deployment**: Vercel-ready

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3001](http://localhost:3001) to view the site.
